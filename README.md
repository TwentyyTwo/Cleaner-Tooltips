<h4 align="center">A simple client-side mod that changes the tooltips of tools and armor to display their attributes as icons.</h4>

## Additional Features
* Compare the attributes of equipped and selected items
* Display durability of tools and armor at three configurable positions
* Display the mining speed of tools like pickaxes and shovels
* Combine the values of equal attributes into a single icon
* Sharpness changes the attack damage attribute

Most of these features can be further customized in the in-game config!

## Compatibility
Cleaner Tooltips should be compatible with most UI mods, but has built-in compatibility for the following mods:
* Apotheosis
* Better Combat
* Legendary Tooltips
